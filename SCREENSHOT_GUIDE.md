## 3️⃣ Phase 8 기능 페이지

### 스크린샷 5: 재무 대시보드
- **파일명**: `05_financial_dashboard.png`
- **URL**: http://139.150.11.99/billing/financial-dashboard
- **설명**: 재무 대시보드 메인 화면
- **촬영 방법**:
  1. 재무 대시보드 접속
  2. 모든 카드와 차트가 로드될 때까지 대기
  3. 페이지 전체 스크롤 캡처 (여러 장 가능)

**주요 요소**:
- 총 수익, 청구 금액, 수금 금액, 수금률
- 미수금 현황 카드
- 현금 흐름 차트
- 기간 선택 드롭다운

### 스크린샷 6: 요금 미리보기
- **파일명**: `06_charge_preview.png`
- **URL**: http://139.150.11.99/billing/charge-preview
- **설명**: 요금 계산기 화면
- **촬영 방법**:
  1. 요금 미리보기 페이지 접속
  2. 샘플 데이터 입력:
     - 거리: 100km
     - 무게: 500kg
     - 팔레트: 2개
  3. "계산" 버튼 클릭
  4. 결과가 표시된 상태에서 캡처
