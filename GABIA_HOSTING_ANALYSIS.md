### 11. 트러블슈팅

#### 가비아 클라우드 Gen2 관련

**문제: 신규 가입 불가**
```
해결: 
- 가비아 계정이 있는 경우: 로그인 후 클라우드 Gen2 신청
- 가비아 계정이 없는 경우: 회원가입 후 클라우드 Gen2 신청
- Gen1은 신규 가입 불가 (2025년 중단)
```

**문제: 비용이 예상보다 높음**
```
확인 사항:
- 서버 사양 (2vCore, 4GB)
- 스토리지 타입 (HDD vs SSD)
- 추가 서비스 (IP, 백업, 로드밸런서)
- 트래픽 사용량 (4TB/월 초과 시 과금)

권장:
- 비용 계산기 사용: https://www.gabiacloud.com/service/estimate/
```

**문제: 성능 부족 (HDD)**
```
해결:
- SSD 타입으로 변경 (추가 비용 발생)
- 또는 Hetzner Cloud로 전환 (NVMe SSD, 저렴)
```
