## 📦 주문 관리

### 주문 목록 조회

**경로**: 좌측 메뉴 > 주문 관리

**화면 구성**:
- 검색 바: 주문번호, 거래처명으로 검색
- 필터: 상태, 온도대, 날짜 범위
- 정렬: 생성일, 희망일자, 주문번호
- 페이지네이션: 페이지당 20건

**주문 상태**:
- 🆕 신규: 새로 생성된 주문
- ⏳ 대기: 배차 대기 중
- ✅ 배차완료: 배차 완료
- 🚛 배송중: 배송 진행 중
- ✨ 완료: 배송 완료
- ❌ 취소: 주문 취소

### 주문 생성

#### 1. 기본 정보 입력

1. **"새 주문" 버튼 클릭**

2. **주문 기본 정보**
   - 주문번호: 자동 생성 또는 수동 입력
   - 거래처 (상차지): 드롭다운에서 선택
   - 거래처 (하차지): 드롭다운에서 선택
   - 제품명: 예) "냉동 닭고기"

3. **온도대 선택** ⚠️ 중요!
   - ❄️ 냉동: -18°C ~ -25°C
   - 🧊 냉장: 0°C ~ 6°C
   - 🌡️ 상온: 실온

4. **물량 정보**
   - 팔레트 수: 예) 10개
   - 중량 (kg): 예) 500kg
   - 부피 (CBM): 선택사항

#### 2. 시간 정보 입력

1. **희망 상차 시간**
   - 날짜 선택
   - 시간 범위 설정 (예: 09:00 ~ 12:00)

2. **희망 하차 시간**
   - 날짜 선택
   - 시간 범위 설정 (예: 14:00 ~ 17:00)

3. **우선순위** (선택)
   - 높음: 긴급 배송
   - 보통: 일반 배송
   - 낮음: 시간 여유 있음

#### 3. 추가 정보 (선택사항)

- 특이사항: 배송 시 주의사항 입력
- 거래처 담당자: 연락처 정보
- 제품 상세: 추가 제품 정보

#### 4. 저장

- "저장" 버튼 클릭
- 주문이 생성되고 목록에 표시됨

### 주문 수정

1. 주문 목록에서 수정할 주문 클릭
2. 상세 정보 화면에서 "수정" 버튼 클릭
3. 정보 수정
4. "저장" 버튼 클릭

⚠️ **주의**: 배차 완료된 주문은 수정 제한이 있을 수 있습니다.

### 주문 삭제

1. 주문 목록에서 삭제할 주문 선택
2. "삭제" 버튼 클릭
3. 확인 메시지에서 "확인" 클릭

⚠️ **주의**: 배차 완료/배송 중인 주문은 삭제할 수 없습니다.

### 주문 검색 및 필터링

#### 검색

- 주문번호로 검색
- 거래처명으로 검색
- 제품명으로 검색

#### 필터

**상태별 필터**:
- 전체
- 신규
- 대기
- 배차완료
- 배송중
- 완료
- 취소

**온도대별 필터**:
- 전체
- 냉동
- 냉장
- 상온

**날짜 범위 필터**:
- 오늘
- 이번 주
- 이번 달
- 사용자 지정 범위

### 엑셀로 내보내기

1. 주문 목록 화면에서 "엑셀 내보내기" 버튼 클릭
2. 날짜 범위 선택
3. "다운로드" 버튼 클릭
4. 엑셀 파일 저장

**파일 형식**: `orders_YYYYMMDD_HHMMSS.xlsx`

## 🚛 배차 관리

### 배차 목록 조회

**경로**: 좌측 메뉴 > 배차 관리

**화면 구성**:
- 배차 목록
- 배차 상태
- 배정된 차량
- 주문 수
- 예상 거리/시간

**배차 상태**:
- 🆕 계획: 배차 계획 단계
- ⏳ 대기: 출발 대기
- 🚛 진행중: 배송 진행 중
- ✅ 완료: 배송 완료
- ❌ 취소: 배차 취소

### AI 자동 배차

#### 1. 자동 배차 시작

1. **"AI 배차 생성" 버튼 클릭**

2. **날짜 선택**
   - 배차할 날짜 선택
   - 여러 날짜 동시 선택 가능

3. **미배차 주문 확인**
   - 선택한 날짜의 미배차 주문 목록 표시
   - 온도대별, 지역별로 그룹화되어 표시

4. **"자동 배차 실행" 버튼 클릭**

#### 2. AI 배차 프로세스

시스템이 다음을 자동으로 수행합니다:

1. **제약 조건 확인**
   - ✅ 온도대 매칭
   - ✅ 팔레트 용량
   - ✅ 중량 제한
   - ✅ 타임 윈도우
   - ✅ 근무시간

2. **최적화 계산**
   - Google OR-Tools VRP 알고리즘 실행
   - 최단 거리 경로 계산
   - 공차 최소화
   - 차량 간 업무 균형

3. **배차 생성**
   - 최적 배차 결과 생성
   - 경로 순서 결정
   - 예상 시간 계산

#### 3. 배차 결과 검토

1. **배차 결과 화면**
   - 생성된 배차 목록
   - 각 배차의 주문 목록
   - 예상 거리/시간
   - 적재율

2. **지도에서 확인**
   - "지도 보기" 버튼 클릭
   - 경로가 지도에 표시됨
   - 순서별로 마커 표시

3. **배차 수정** (필요시)
   - 주문 추가/제거
   - 순서 변경
   - 차량 변경

4. **배차 확정**
   - "확정" 버튼 클릭
   - 기사에게 자동 알림 발송

### 수동 배차

#### 1. 배차 생성

1. **"수동 배차" 버튼 클릭**

2. **차량 선택**
   - 가용 차량 목록에서 선택
   - 온도대 확인

3. **주문 추가**
   - "주문 추가" 버튼 클릭
   - 미배차 주문 목록에서 선택
   - 온도대가 맞는 주문만 표시됨

4. **경로 순서 설정**
   - 드래그 앤 드롭으로 순서 변경
   - 시스템이 자동으로 거리/시간 재계산

#### 2. 배차 검증

시스템이 자동으로 검증:

- ✅ 팔레트 초과 여부
- ✅ 중량 초과 여부
- ✅ 타임 윈도우 준수
- ✅ 근무시간 초과 여부

⚠️ 경고 또는 오류 메시지 표시 시 수정 필요

#### 3. 저장 및 확정

1. **"저장" 버튼**: 임시 저장
2. **"확정" 버튼**: 배차 확정 및 알림 발송

### 배차 수정

1. 배차 목록에서 수정할 배차 선택
2. "수정" 버튼 클릭
3. 주문 추가/제거 또는 순서 변경
4. "저장" 버튼 클릭

⚠️ **주의**: 진행 중인 배차는 수정 제한

### 배차 취소

1. 배차 목록에서 취소할 배차 선택
2. "취소" 버튼 클릭
3. 취소 사유 입력
4. "확인" 버튼 클릭

⚠️ **주의**: 완료된 배차는 취소할 수 없습니다.

- ⚠️ 경고: 온도가 범위를 벗어남
- 🔔 자동 알림: 담당자에게 즉시 알림
- 🔴 지도에 빨간색으로 표시

**온도 이력 확인**:

3. 시간별 온도 그래프 확인

**ETA 계산**:
- 실시간 교통 정보 반영
- 남은 거리 기반
- 평균 속도 고려

**ETA 표시**:
- 차량 목록에 표시
- 지도 팝업에 표시
- 15분 단위로 업데이트

### 실시간 알림

**알림 유형**:
- 🔔 온도 이상
- 📍 목적지 도착
- ⚠️ 지연 예상
- ✅ 배송 완료

**알림 수신**:
- 브라우저 푸시 알림
- 시스템 내 알림 센터
- 이메일 (설정 시)

## 🏢 거래처 관리

### 거래처 목록

**경로**: 좌측 메뉴 > 거래처 관리

**화면 구성**:
- 거래처 목록
- 검색 및 필터
- 지도 보기

**거래처 구분**:
- 🏭 상차지: 물건을 싣는 곳
- 🏢 하차지: 물건을 내리는 곳
- 🔄 양쪽: 상차/하차 모두 가능

### 거래처 등록

#### 1. 기본 정보 입력

1. **"거래처 추가" 버튼 클릭**

2. **기본 정보**
   - 거래처 코드: 고유 코드 (예: CL001)
   - 거래처명: 정식 명칭
   - 거래처 구분: 상차지/하차지/양쪽

3. **주소 정보**
   - 도로명 주소 입력
   - "주소 검색" 버튼으로 우편번호 찾기
   - 상세 주소 입력

4. **자동 지오코딩**
   - 주소 입력 후 자동으로 좌표 계산
   - 지도에 위치 표시
   - 위치 확인 후 저장

#### 2. 운영 정보 입력

1. **운영 시간**
   - 시작 시간: 예) 09:00
   - 종료 시간: 예) 18:00
   - 점심시간: 12:00 ~ 13:00 (선택)

2. **시설 정보**
   - 지게차 유무: 예/아니오

## 🚙 차량 관리

### 차량 목록

**경로**: 좌측 메뉴 > 차량 관리

**화면 구성**:
- 차량 목록
- 상태별 필터
- 온도대별 필터

**차량 상태**:
- 🚛 가동: 배송 중
- 💤 대기: 배차 대기
- 🔧 정비: 정비 중
- ❌ 휴차: 사용 불가

### 차량 등록

#### 1. 기본 정보

1. **"차량 추가" 버튼 클릭**

2. **차량 정보**
   - 차량 번호: 예) 서울12가3456
   - 차량 종류: 윙바디/탑차/냉동탑차
   - 연식: 제조 연도

3. **온도대 설정** ⚠️ 중요!
   - ❄️ 냉동: -18°C ~ -25°C 전용
   - 🧊 냉장: 0°C ~ 6°C 전용
   - 🔄 겸용: 냉동/냉장 모두 가능
   - 🌡️ 상온: 상온 운송

#### 2. 적재 정보

1. **적재 용량**
   - 팔레트 수: 최대 적재 가능 팔레트
   - 최대 중량 (kg): 최대 적재 중량
   - 부피 (CBM): 적재 공간 부피

2. **예시**:
   - 5톤 차량: 팔레트 10개, 5000kg
   - 1톤 차량: 팔레트 4개, 1000kg

#### 3. 운영 정보

1. **기사 배정**
   - 주 기사: 주로 운행하는 기사
   - 예비 기사: 대체 가능한 기사

2. **UVIS 연동**
   - 단말기 ID: UVIS 단말기 번호
   - 차량 ID: UVIS 시스템 차량 ID

3. **정비 정보**
   - 다음 정비일: 정기 점검 예정일
   - 정비 주기: 예) 6개월마다

#### 4. 저장

- "저장" 버튼 클릭
- 차량이 생성되고 목록에 표시됨

## 🆘 문제 해결

### 로그인 문제

**문제**: 로그인이 안됩니다.

**해결**:
1. ✅ Username/Password 대소문자 확인
2. ✅ Caps Lock 꺼져 있는지 확인
3. ✅ 브라우저 캐시 삭제
4. ✅ 시크릿 모드로 시도
5. ✅ 비밀번호 재설정 사용

### 지도 문제

**문제**: 지도가 표시되지 않습니다.

**해결**:
1. ✅ 네이버 지도 API 키 확인
2. ✅ 브라우저 위치 권한 허용
3. ✅ 페이지 새로고침
4. ✅ 다른 브라우저로 시도

### 실시간 업데이트 문제

**문제**: 실시간 업데이트가 안됩니다.

**해결**:
1. ✅ WebSocket 연결 확인
2. ✅ 방화벽 설정 확인
3. ✅ 브라우저 최신 버전 사용
4. ✅ 페이지 새로고침

### 엑셀 업로드 문제

**문제**: 엑셀 파일 업로드가 실패합니다.

**해결**:
1. ✅ 템플릿 형식 확인
2. ✅ 파일 크기 10MB 이하
3. ✅ 인코딩 UTF-8 사용
4. ✅ 필수 항목 모두 입력
5. ✅ 주소 정확히 입력

### AI 배차 문제

**문제**: AI 배차가 생성되지 않습니다.

**해결**:
1. ✅ 미배차 주문 존재 확인
2. ✅ 가용 차량 확인
3. ✅ 온도대 매칭 확인
4. ✅ 타임 윈도우 검토
5. ✅ 로그 확인

### 온도 모니터링 문제

**문제**: 온도 정보가 업데이트되지 않습니다.

**해결**:
1. ✅ UVIS 연동 상태 확인
2. ✅ 차량 단말기 작동 확인
3. ✅ API 키 설정 확인
4. ✅ 네트워크 연결 확인

### 성능 문제

**문제**: 시스템이 느립니다.

**해결**:
1. ✅ 브라우저 캐시 삭제
2. ✅ 탭 개수 줄이기
3. ✅ 불필요한 확장 프로그램 비활성화
4. ✅ 페이지 새로고침
5. ✅ 시스템 관리자에게 문의
