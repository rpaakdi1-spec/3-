## 🎯 시스템 개요

### 시스템 목적

Cold Chain Dispatch System은 냉동·냉장 화물의 효율적인 배차와 실시간 추적을 지원하는 AI 기반 통합 관리 시스템입니다.

### 주요 특징

- ✅ **AI 기반 자동 배차**: 최적의 차량과 기사 자동 배정
- ✅ **실시간 위치 추적**: GPS 기반 배송 현황 모니터링
- ✅ **교통 정보 연동**: 실시간 교통 상황 반영
- ✅ **배송 추적**: 고객용 배송 추적 링크 제공
- ✅ **성능 분석**: 배차 효율성 및 KPI 대시보드

### 접속 방법

```
웹 접속: https://your-domain.com
사용자 ID와 비밀번호로 로그인
```

## 👥 사용자 역할

### 1. 관리자 (Admin)
- 시스템 전체 관리
- 사용자 계정 관리
- 거래처 및 차량 관리
- 시스템 설정 변경

### 2. 배차 담당자 (Dispatcher)
- 주문 등록 및 관리
- 배차 생성 및 수정
- 배송 현황 모니터링
- 긴급 재배차 처리

### 3. 기사 (Driver)
- 배차 확인
- 배송 상태 업데이트
- GPS 위치 공유
- 배송 완료 처리

### 4. 거래처 (Client)
- 주문 현황 조회
- 배송 추적
- 배송 이력 확인

## 🚀 주요 기능

### 1. 주문 관리

#### 주문 등록
1. 좌측 메뉴에서 **"주문 관리"** 클릭
2. **"신규 주문"** 버튼 클릭
3. 주문 정보 입력:
   - 거래처 선택
   - 상차지 주소 입력
   - 하차지 주소 입력
   - 화물 정보 (온도, 무게, 부피)
   - 희망 배송 시간
4. **"저장"** 버튼 클릭

#### 주문 조회
- 주문 목록에서 검색 필터 사용:
  - 날짜 범위
  - 거래처
  - 상태 (대기/진행중/완료/취소)

#### 주문 수정/취소
- 주문 목록에서 해당 주문 클릭
- **"수정"** 또는 **"취소"** 버튼 클릭
- 주의: 배차 완료 후에는 수정 불가

### 2. 배차 관리

#### 자동 배차
1. **"주문 관리"** 메뉴에서 배차 대기 중인 주문 선택
2. **"자동 배차"** 버튼 클릭
3. AI가 최적의 차량과 기사를 자동으로 배정
4. 배차 결과 확인 후 **"배차 확정"** 클릭

#### 수동 배차
1. **"주문 관리"** 메뉴에서 주문 선택
2. **"수동 배차"** 버튼 클릭
3. 차량 및 기사 직접 선택
4. 배차 정보 입력:
   - 출발 예정 시간
   - 도착 예정 시간
   - 특이사항
5. **"배차 확정"** 버튼 클릭

#### 배차 현황 모니터링
- **"배차 현황"** 메뉴에서 실시간 배송 상태 확인
- 지도에서 차량 위치 실시간 확인
- 배송 단계별 상태:
  - 🟡 배차 완료 (출발 전)
  - 🔵 상차지 이동 중
  - 🟢 하차지 이동 중
  - ✅ 배송 완료

#### 긴급 재배차
1. **"배차 현황"** 메뉴에서 재배차가 필요한 배차 선택
2. **"재배차"** 버튼 클릭
3. 재배차 사유 입력
4. 새로운 차량/기사 선택
5. **"재배차 확정"** 클릭

### 3. 차량 관리

#### 차량 등록
1. **"차량 관리"** 메뉴 클릭
2. **"차량 등록"** 버튼 클릭
3. 차량 정보 입력:
   - 차량 번호
   - 차량 유형 (냉동/냉장/냉동+냉장)
   - 적재 용량 (톤)
   - 온도 범위
4. **"저장"** 버튼 클릭

#### 차량 상태 관리
- 차량 목록에서 상태 변경:
  - 🟢 가용 (운행 가능)
  - 🔴 불가 (정비 중, 고장 등)
  - 🟡 배차 중

### 4. 거래처 관리

#### 거래처 등록
1. **"거래처 관리"** 메뉴 클릭
2. **"거래처 등록"** 버튼 클릭
3. 거래처 정보 입력:
   - 거래처명
   - 사업자 번호
   - 담당자 정보
   - 주소 및 연락처
4. **"저장"** 버튼 클릭

### 5. 배송 추적

#### 고객용 배송 추적
1. 배차 완료 시 자동으로 추적 링크 생성
2. 고객에게 추적 링크 전송:
   ```
   https://your-domain.com/tracking/ABC123DEF456
   ```
3. 고객은 별도 로그인 없이 실시간 배송 현황 확인 가능

#### 추적 정보
- 현재 배송 단계
- 차량 실시간 위치 (지도)
- 예상 도착 시간 (ETA)
- 배송 기사 연락처
- 배송 이력

### 6. 통계 및 리포트

#### 대시보드
- **"대시보드"** 메뉴에서 주요 지표 확인:
  - 금일 배차 현황
  - 월간 배송 통계
  - 차량 가동률
  - 거래처별 배송량

#### 리포트 생성
1. **"통계/리포트"** 메뉴 클릭
2. 리포트 유형 선택:
   - 일일 배송 리포트
   - 월간 실적 리포트
   - 거래처별 리포트
   - 차량별 리포트
3. 기간 설정
4. **"리포트 생성"** 버튼 클릭
5. Excel 또는 PDF로 다운로드

## ❓ 자주 묻는 질문 (FAQ)

### Q1. 비밀번호를 잊어버렸어요.
**A**: 로그인 화면에서 "비밀번호 찾기"를 클릭하고, 등록된 이메일 주소를 입력하세요. 비밀번호 재설정 링크가 이메일로 발송됩니다.

### Q2. 배차를 취소하고 싶어요.
**A**: 
- 배차 완료 후 출발 전: 배차 현황에서 해당 배차의 "취소" 버튼 클릭
- 이미 출발한 경우: "재배차" 기능을 사용하여 다른 차량으로 변경

### Q3. 자동 배차가 실패했어요.
**A**: 자동 배차 실패 원인:
- 가용한 차량이 없는 경우
- 차량 용량이 부족한 경우
- 온도 조건에 맞는 차량이 없는 경우

해결 방법:
1. 차량 상태 확인 (정비 중인 차량이 있는지)
2. 수동 배차로 직접 차량 선택
3. 배차 시간을 조정하여 재시도

### Q4. 배송 추적 링크가 작동하지 않아요.
**A**: 
- 배차가 완료된 후에만 추적 링크가 활성화됩니다
- 링크 형식: `https://your-domain.com/tracking/[추적번호]`
- 추적번호가 정확한지 확인하세요
- 문제가 계속되면 관리자에게 문의하세요

### Q5. 주문을 대량으로 등록하고 싶어요.
**A**: 
1. 주문 관리 화면에서 "엑셀 다운로드" 버튼 클릭
2. 다운로드된 템플릿 파일에 주문 정보 입력
3. "엑셀 업로드" 버튼으로 파일 업로드
4. 오류가 있는 경우 오류 목록 확인 후 수정

### Q6. 차량의 실시간 위치가 업데이트되지 않아요.
**A**: 
- 기사 앱에서 위치 권한이 허용되어 있는지 확인
- 기사의 스마트폰 GPS가 켜져 있는지 확인
- 네트워크 연결 상태 확인
- 문제가 계속되면 기사에게 앱을 재시작하도록 요청

### Q7. 리포트를 생성할 수 없어요.
**A**: 
- 데이터가 있는 기간을 선택했는지 확인
- 브라우저의 팝업 차단 설정 확인
- 다운로드 폴더의 용량 확인
- 다른 브라우저로 시도해보세요

### Q8. 시스템이 느려요.
**A**: 
- 브라우저 캐시를 삭제하세요
- 사용하지 않는 탭을 닫으세요
- 권장 브라우저: Chrome, Edge, Firefox (최신 버전)
- 인터넷 연결 속도 확인
- 문제가 계속되면 시스템 관리자에게 문의하세요
